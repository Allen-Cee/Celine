# 雕龙｜小王子：愿二十年后绮梦浮生比青春更狠

【2015-11-04】

【原文链接：https://mp.weixin.qq.com/s/AD1GkJOznYK1uvWjqIc8IA】

电影院灯光亮起的时候我还在掉泪，越擦越多。很多人坐在原地不想走，定在那看着巨大的灯光下屏幕上一帧帧闪过的《小王子》原书的内页，哪怕工作人员在出口一遍又一遍徒劳地喊着“没有彩蛋”。

很多电影会让你忘记现实，在超级英雄们健硕的肌肉和光怪陆离的星际穿越中你仿佛成为主宰世界的国王；但是有些电影帮你温暖现实，提醒你在生理痛的时候你需要的不是热水而是一碗加红糖的姜汤。

出于处女座的一丝不苟，我一般在看电影前一定要查清楚它的导演编剧演员配乐剧情乃至评测，非要把它挖个底朝天不可。但小王子，乍一听说要上映时，我就大喊着“约约约”，为它投票，仓皇着推搡着就坐在了电影院里，而此时的我甚至连它是不是根据原著都不知道，就带上了3D眼镜。

大抵是儿时的一点情结。正如我在朋友圈发的：“最近颇觉诸事压身有如梦魇，需要约一只小王子以求治愈。”你刚踏入成人世界在里面浮浮沉沉，刚好，小王子就出现了，像一艘发自你童年的船，你别无选择，心甘情愿地躲进去，遮风避雨。

对情节的未知造成一种恐惧，当影片开始航拍模型显示出一个现代化的城市并且不断显像的时候，我心里一咯噔：糟了，好像真的是改编。

这种惴惴却很快地在小女孩进入面试房间后弥散于无形，因为导演故意采取的小孩子的视角是那样可爱，那些面容模糊在黑暗里的主考官，犹如鬼魅的暗绿色光线，还有机械划一的翻页点头，都透露出在孩子的世界恶意丑化大人的一点得逞的狡黠。

真正让我进入电影的还是妈妈对鸽子发起的连环水弹攻击，一个对生活充满斗志总是一身职业装笔挺的妈妈，和一个懵懵懂懂却努力表现出成熟的女儿，无论如何，这样的组合总是令人期待的。而一个不会忘记提起鸽子和风筝导演，他讲述的故事应该也不会太无趣，不是吗？

&nbsp;

大家还记得那只会功夫的熊猫吗？一贯善于荧幕叙事的导演马克奥斯本，这次给我们带来的《小王子》出现了一些可称之为惊喜的突破——CG＋定格的混搭风格。CG动画天生的像素感，特别适合戏中小女孩被标准化的和冷漠的现实世界；而定格动画的“手作感”，那种质朴的稍显笨拙的感动，犹如切黄油的餐刀恰如其分地切入我们内心深处，恰好能给小王子的世界带来特有的属于人的温度。据说导演找来了皮克斯与梦工厂团队，用3D、手绘以及定格技巧，把小王子的形象分别用折纸静态和三维动画都表现了一遍。片中只要拍到小王子的部分，就会有大段大段的留白，仅有画面和汉斯季默柔肠百结的配乐。这种类似默片的方式为观众还原了一个如此逼真的小王子。马克奥斯本在谈到剧本处理时说，“面对死亡和失去，试图在感情上做缓和处理…希望观众是笑着走出影院。”也许是他作为动画导演出身的一点小小的柔情。

我想我对导演马克奥斯本的评价应该是中肯的。在我看到荧幕上纸质的小王子出现的时候更是确认这一点。还有什么比用纹路纵横的纸张来还原小王子更恰切的设计？在我们的眼睛已经适应了皮克斯迪士尼公司精准建模绘制出来的光滑完美的小人儿的时候，这种触手可及的细腻质感仿佛一个古老的咒语一下子击中了我的心。几乎是在瞬间我回忆起了小时候躺在妈妈身边一手一页一页地翻着小王子的纸质书，另一边用手剥开太妃糖时，那独特的、在舌尖留下挥之不去的浓醇香。

儿时的时间总是绵长得好似棉花糖机上的糖丝，牵牵绕绕细密地将你裹在里面，仿佛就可以这样一直到永远。

&nbsp;

可是时间真的会为了某个人的一厢情愿而停滞不前吗？你还沉浸在小王子的童话的时候，却看见了时光的齿轮忠实转动时出现的那个王子先生。你看见了他被烟囱弄的脏污的脸颊，看见了他飘逸的长围巾被改成了领带拘束地绑在胸前，你看见他身量拔高像一株不合时宜的竹竿在成人的世界艰难地摇来晃去。

你觉得这不对劲。可你说不出这样不对劲在哪里。

就好像你在看到童话破碎后小女孩被妈妈锁在屋里做习题，密密麻麻的3a＋4b＋5c。就好像你看到小狐狸被丢在垃圾桶的最底层，一袋一袋垃圾丢进去直至那抹橙黄彻彻底底地不见天日。

&nbsp;

你觉得胸口闷闷的，有眼泪要流，却不知道要从哪里流出来。

“真正的问题不在于长大，在于遗忘。”

于是轰的一声。

你的眼泪终于狼奔豕突找到出口，如愿倾泻。

我们害怕的是什么。我们不安的是什么。我们在逐渐惯常的成人世界里行走，却偶尔会感觉不适意的是什么。

是遗忘啊。

&nbsp;

你记得今天晚上要去理科楼开会，你记得三天后是一篇稿件的DDL，你甚至记得下个月11号是你的专业课考试。

可是你还记得小时候一毛五分钱的冰棒的味道吗，你还记得当年中考后说好的要每年相聚的朋友吗，你还记得小时候在作业下面偷偷垫上一本喜欢的书，徜徉在光怪陆离的世界里那种隐秘的快乐吗。

成年人就像趋利避害的草履虫，在童年的回忆和现实的压力之中权衡了一下，然后毅然决然地决定，嗯，还是后者更重要些，然后在飞速游动的过程中慢慢地慢慢地忘记了那一整个色彩斑斓的童年。

当我们开始寻求，我们就已经失去；而我们不开始寻求，我们根本无法知道自己身边的一切是如此可贵。

是你为玫瑰付出的时间让她变得如此重要。可是有一天，你不愿意再为她付出时间了，她就在你的心里慢慢枯死，直到连带着那半亩花田一起彻彻底底地消失在你心里。

我们现在有多少自诩成功精英的大人，是心里没有这半亩花田的人呢。

所以你不会再为了小王子感动了。所以你甚至觉得他可笑和无聊。

你看到小孩子的时候会觉得惊恐。为什么呢？因为他们会提醒你，你已经忘记小时候幻想中的英雄凛凛的自己了。你过上了与理想的自己渐行渐远的生活。而与自己的理想背道而驰的人是脆弱的。其实你害怕的并不是小孩，你只是害怕思考这个缺乏安全感的问题：为什么我把我的生活过成了我不想要的模样？规律安逸的生活已经驯化了你的思维，你情愿沉溺在日程表日复一日的闹铃里而放弃自己的思考能力。质疑这种按部就班的生活会让你不安。没有什么比内心的冲突更让人痛苦的了。

小时候我们都像桀骜不驯的普罗米修斯一样企图拯救世界。但你终会发现其实我们只是被巨鲸吞入腹中的约拿。耶和华将桀骜不羁的约拿投入巨鲸腹中，约拿对抗了三天三夜，最终以归顺而得拯救。

代价是什么呢。

你把那个鲜活的自己，毫不心疼地放进了碎纸机里。

“咔”的一声。获得拯救。

&nbsp;

愿你拥有属于自己的花田。

——我是说，如果你还想在胸口尚存一丝鲜活的话。

希望你还记得自己的花田，那里只种着一株玫瑰，你仰头可以看见无数笑声洒下来的星星。

时光如河。童年的人早已随波逐浪而去，成年的人偶尔涉一涉那河，在温柔不变的河水里依稀感受着过去的体温。

无论你是痴立水中不得上岸，抑或是隔岸行走偶尔湿一湿鞋。

总之你记得有这样一条温暖的河。它总在那里，伴你一生起落。

浮生绮梦。未若青春狠。
