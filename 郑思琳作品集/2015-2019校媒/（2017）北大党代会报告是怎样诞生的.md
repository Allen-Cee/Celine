# 幕后｜北大党代会报告是怎样诞生的

【2017-11-19】

【原文链接：https://mp.weixin.qq.com/s/s2nFKbjZJS5RTWeufOBdbw】

2017年11月18日，北京大学百周年纪念讲堂。中国共产党北京大学第十三次党员代表大会在这里隆重举行。

校党委书记郝平代表上届党委，向大会作了题为《以党的十九大精神为指引 开启中国特色世界一流大学发展新征程》的报告。现场一次又一次响起热烈的掌声。

这是北大共产党人的铮铮誓言——

以党的十九大精神为指引，全面贯彻习近平新时代中国特色社会主义思想，落实立德树人根本任务，加快推进“双一流”建设，开启中国特色世界一流大学发展新征程，为实现“两个一百年”奋斗目标和中华民族伟大复兴的中国梦作出新的历史性贡献。

这是北大人在光荣传统中的庄严使命——

为人民谋幸福，为民族谋复兴。

这是北大人在新时代中的奋发有为——

我们要聚焦七大任务，加快“双一流”建设；要突出五大重点，全面深化综合改革；更要坚持“打铁必须自身硬”，开创党建和思想政治工作新局面。

开启中国特色世界一流大学发展的新征程，力争早日实现跻身世界一流大学前列的战略目标——北大第十三次党代会报告起草工作始终在北大党委的直接领导下进行，凝聚了广泛共识，是全校党员和全体师生员工智慧的结晶。

这篇引发无数阅读与讨论的报告是如何诞生的？报告中又有哪些意义深远的重要论述呢？北大官微专访了党委工作报告起草小组成员、党委政策研究室的有关同志，为你揭秘北大党代会报告背后的故事。

## 一、诞生记

党代会报告是如何产生的

&nbsp;

Q（代记者）：这次党代会报告的前期准备大致经历了什么样的过程？

A（代起草小组）：经学校党委研究，并与教育部党组和北京市委教育工委沟通，拟定于2017年11月召开中共北京大学第十三次党代会。

&nbsp;

6月

校党委明确由党委政研室牵头，会同党办校办、纪委办公室、医学部两办等单位，组成第十三次党代会筹备工作起草组，在深入调查研究、广泛听取各方面意见的基础上启动起草工作。

&nbsp;

8月28日

北京大学第十三次党代会动员部署会。郝平在会上作了题为《扎实筹备并开好第十三次党代会，以优异成绩迎接党的十九大胜利召开》的动员报告。

按照动员部署会的要求，起草组围绕学校一系列重大理论和实践问题开展实地调研、广泛征求意见。

&nbsp;

9月4日

学校党委向全校党员、师生员工、校友和社会各界人士发出《关于北京大学第十三次党代会报告起草工作征求意见的通知》，面向全校党员、师生员工、校友以及社会各界人士征求意见建议。

&nbsp;

9月7日至10月25日

学校党委分别召开了机关职能部门代表座谈会、各民主党派、侨联负责人及无党派人士代表座谈会、离退休教职工代表座谈会和学生代表座谈会，纪委办公室组织召开各单位纪检委员座谈会和纪委委员全会，后勤党委和机关党委、校工会还分别召开了后勤系统干部职工代表座谈会和机关青年职员代表座谈会，听取了相关方面代表的意见。

&nbsp;

9月至11月

校党委书记郝平带队，先后到新闻与传播学院、新媒体研究院、城市与环境学院、社会学系、物理学院、生命科学学院、信息科学技术学院、人文社会科学研究院、法学院、中国语言文学系、口腔医院、北京大学附属中学、燕园街道、校办产业党工委、北京大学附属小学、外国语学院、地球与空间科学学院、化学与分子工程学院、工学院、数学科学学院、国际关系学院、校友工作办公室等单位进行深入调研。在调研过程中，广泛听取了各院系、各单位对党委工作报告的意见建议。

&nbsp;

如何对过去5年校党委的工作、创建中国特色世界一流大学所取得的主要成就和经验进行全面客观的总结？

如何概括未来5年学校发展面临的形势、机遇与挑战，工作中的主要困难和问题？

如何阐述新形势下校党委工作的目标、指导思想和主要任务，学校加快创建中国特色世界一流大学的形势任务、指导思想、阶段目标、发展战略等重大问题，特别是围绕“建设什么样的北大、怎样建设这样的北大”这一根本问题，如何谋划学校今后五年及更长一个时期的发展？

如何实现今后5年校党委、校纪委以及学校改革发展稳定？

……

&nbsp;

过去5年，党委把方向、管大局、做决策、保落实，确保中央大政方针和决策部署在北大落地生根，形成生动实践。

过去5年，综合改革蹄疾步稳，重要领域和关键环节改革取得突破性进展。

过去5年，推动学校科学发展，办学实力和影响力显著增强。

过去5年，全面从严治党层层推进，党建和思想政治工作得到进一步加强。

&nbsp;

春华秋实，硕果累累。

&nbsp;

Q：在党代会报告起草之前，为什么要经历这么多准备工作呢？

A：校党委提出，党代会报告的起草工作必须充分发扬民主，群策群力；充分调动全体党员的积极性主动性创造性，落实好党的群众路线，凝聚广大师生员工的共识，使报告的起草过程成为统一思想、统一认识、振奋士气、鼓舞人心的过程。前期准备工作的周密，正是报告充分反映民生、表达民情的体现。

&nbsp;

Q：我们也发现，在整个前期准备过程中有很多体现出意见收集的部分。例如在校园网上发布《关于北京大学第十三次党代会报告起草工作征求意见的通知》。我们很好奇，这些征集成效如何呢？

A：征集帖于9月4日发布，截至11月1日，起草组共向全校204位各方面代表呈送了征求意见稿，收到信件、花脸稿40余份，参加各方面代表座谈会8次，完成单独访谈20余次，累计人数400多人次，搜集到各类意见建议500余条。起草组对提出意见、建议的领导、老师和同学都给予积极回复，表达真诚感谢，并认真消化吸收，汇总形成书面文件，呈学校党委参阅。

&nbsp;

Q：都征集到了一些怎样的意见？

A：征集到的意见方方面面，我们大致归类为题目和主题、发展目标和阶段、成绩总结、问题与反思、历史回顾与总结、总体思路、人才培养、人才队伍建设、学术氛围、学科布局、内涵发展、本部与医学部深度融合、医学教育、附属医院、空间资源、国际化、管理队伍和后勤教辅队伍、发挥离退休教师作用、民生问题、党建、关键节点、报告篇幅、行文风格等23个方面。

除了广泛征求意见之外，校党委要求全校各单位积极行动起来，认真总结过去五年的工作，对未来五年的工作开展研讨，提出科学的规划方案。在这样的指导下，各院系党委通过召开专题民主生活会等形式，倾听师生员工的意见，找差距、补短板，总结过去、梳理经验、谋划未来，共同为党代会的顺利召开打好基础。

&nbsp;

## 二、启示录

党代会报告，我们怎么读？

&nbsp;

Q：报告中提到，大会的主题是:以党的十九大精神为指引，全面贯彻习近平新时代中国特色社会主义思想，落实立德树人根本任务，加快“双一流”建设，开启中国特色世界一流大学发展新征程，为实现“两个一百年”奋斗目标和中华民族伟大复兴的中国梦作出新的历史性贡献。您如何解读这个主题呢？

A：本届党代会召开正好在党的十九大胜利闭幕后不久。在这个特殊的时间段、新时期的序幕上，党代会报告紧跟时代思潮，充分贯彻落实十九大精神和习近平新时代中国特色社会主义思想，指导北大建设工作的展开。体现在报告中，我们可以看到，第一点第一条就是“把方向、管大局、做决策、保落实，确保中央大政方针和决策部署在北大落地生根”。其中也强调了在北大要牢固树立“四个意识”，深入贯彻党中央治国理政新理念新思想新战略；要掀起学习宣传十九大精神的热潮；要深入贯彻落实2014年5月4日习近平总书记在我校师生座谈会上的重要讲话精神；要深入贯彻落实全国高校思想政治工作会议和中央31号文件精神；要积极配合中央巡视，坚决整改落实；要深入学习贯彻习近平总书记对北京的重要讲话精神，立足北京、服务北京、融入北京。这些都是大会主题的体现。

&nbsp;

Q：我们注意到，报告共有五个部分，其中有两个部分提到了回顾历史和北大的传统。这也是报告要着重强调的内容吗？

A：是的。相信听过报告的人都有所体会，报告用了大篇幅总结了北大的光荣革命传统，系统回顾了近120年来的发展道路，高度肯定了历次党代会和历届学校党政领导班子的贡献。

之所以要强调这一点，是由于报告中提出的：我们要深刻认识并正确处理“创建世界一流大学”与“为实现中华民族伟大复兴中国梦做出新的历史性贡献”之间的辩证关系。

我们应当认识到，从根本上说，两者是一致的，建设世界一流大学越有成效，就越能够为国家民族作出更大贡献。但是，两者又不能等同和混淆，前者是手段，后者是目的。如果我们不能把目的与手段区分开来，就会跟在他人后面亦步亦趋，依样画葫芦，就不可能办出自己的特色，不可能办好中国的世界一流大学。因此，我们应当不忘历史，在总结历史经验中摸索出适合北大的发展道路。

&nbsp;

Q：“双一流建设”确实也是本次大会的关键词之一。报告中是如何体现这一点的呢？

A：报告的第四部分专门提到了“突出五大重点，全面深化综合改革”。报告指出，围绕“双一流”建设七大任务，要重点推进五大改革：

1. 进一步深化人才培养模式改革

2. 进一步深化人事制度改革

3. 进一步深化治理体系改革

4. 进一步深化学术体系改革

5. 进一步深化资源配置体系改革

同时，还有很重要的一点就是，报告注意到了要“根据不同学科的特点确立了不同的建设任务”。总的来看，报告对于“双一流”建设的部署可以说是各有侧重，全面兼顾。

&nbsp;

Q：除了第四部分的末尾提到要根据不同学科的特点确立不同的建设任务，其他四个部分也在末尾部分对北大的具体现实有所关照。这一点也和报告前期广泛的调研准备工作有关吗？

A：可以这么说。报告是在充分吸收各方面意见的基础上，经过十余轮的修改形成的，力求充分发扬民主，落实好党的群众路线。正是如此翔实严密的前期准备工作，才使报告能够充分陈情达意。虽然报告对北大建设的各方面做出了高屋建瓴的指导，但每个部分最后会回归民生，贴近民生关注的问题，落实到师生学习生活工作环境的改善。这是难能可贵的一点。

&nbsp;

Q：总结来看，您觉得这份党代会报告有哪些特点呢？

A：总的来说，这是一份高举旗帜、主题鲜明的报告；这是一份发扬传统、继往开来的报告；这是一份谋篇布局、科学规划的报告；这也是一份求真务实、师生为本的报告，这是一份团结奋进、同舟共济的报告；这是一份真诚厚道、很接地气的报告。

中共中央政治局委员、北京市委书记蔡奇同志说，这是一个求真务实、凝心聚力的好报告。
