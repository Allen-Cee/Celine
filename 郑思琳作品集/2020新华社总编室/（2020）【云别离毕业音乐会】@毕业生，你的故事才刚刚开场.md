# @毕业生，你的故事才刚刚开场……

【2020-07-10】

【原文链接：https://mp.weixin.qq.com/s/ZP1s0mPvz-EzLtLTM4r08w】

&nbsp;

“长亭外，古道边，芳草碧连天……”

8日晚“2020毕业音乐会”上

随着天南海北的毕业生

和音乐人们共同深情合唱的《送别》

一张张新华社老照片

带着毕业生们走进

各自母校的青涩年华

&nbsp;

1935年

武汉大学新建图书馆

如今虽然已是毕业生口中的“老图”

但它一直是大学精神的青春象征

&nbsp;

清华、上交大……

老照片里

有一代代毕业生的青春岁月

一张张年轻面孔朝气蓬勃

泛黄的照片也难掩学子们的神采飞扬

&nbsp;

无论是以前，还是现在

毕业照表现的是圆满的团聚

象征的却是长远的分离

在毕业的这一天

是同样矛盾着的欢呼和泪流满面

&nbsp;

而在这分别的前夜

在这场特别的毕业音乐会上

西湖大学校长施一公

北京大学常务副校长詹启敏

也为广大毕业生们送来祝福和嘱托

&nbsp;

黄轩、沈腾、张靓颖、胡夏等学长、学姐

也用自己的亲身体验和深情的歌声

为本届毕业生们送上了

最真挚的寄语和精彩的演出

&nbsp;

今年，高校毕业生达到874万人

在1949年新中国成立时

高校毕业生仅有2.1万人

&nbsp;

71年来

中国高等教育实现了

从精英化向大众化的历史跨越

一代代毕业生用青春

在中国大地上书写时代的答卷

&nbsp;

今年

在抗击新冠肺炎疫情的严峻斗争中

年轻的面孔已经担起大任

北京大学援鄂医疗队中

最年轻的队员刘金鹏

在前线度过了23岁生日

&nbsp;

浙江大学医学院附属第二医院护士余赛婷

瞒着爸妈偷偷报名驰援湖北

她说

“这是23年来最勇敢的决定。”

还有一批毕业生正要奔赴边疆

中国石油大学（北京）克拉玛依校区118名毕业生

选择到新疆基层工作

他们却说

“只有荒凉的沙漠，没有荒凉的人生。”

&nbsp;

这届高校毕业生经受了磨练、收获了成长

也切身体会到

“志不求易者成，事不避难者进”

青年一代有理想、有本领、有担当

国家就有前途，民族就有希望

&nbsp;

同学们

互道珍重后

让我们

以青春为笔，岁月为卷

不畏艰难险阻，勇担时代使命

写下属于我们的

新时代新篇章
