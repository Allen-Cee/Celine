# 是他，让村民“羊”起了笑脸

【2020-10-24】

【原文链接：https://mp.weixin.qq.com/s/yVaC6Ikft-QW1XztohyNYw】

&nbsp;

这里曾产业匮乏

村民靠几亩薄田勉强糊口

一家人一整年只有几千元收入

&nbsp;

这里，是西藏日喀则市尼辖乡宗措村

在藏语里的意思

就是向阳而生

&nbsp;

宗措村驻村第一书记旺青罗布的到来

彻底改变了这里曾经的面貌

带领村民真正地“向阳而生”

他始终坚信

“人跟人之间的感情

能够融化最厚实的冰雪

也能够征服世界上最高的山峰”

&nbsp;

是什么

支撑他跨越一座座山峰

虽千里犹无悔？

&nbsp;

是什么

让他把一分一厘

积攒成老百姓的希望？

&nbsp;

是什么

让一张张黝黑的面容上

绽放最灿烂的笑容？
