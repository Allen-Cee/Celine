# 这一切，比我能想象的还好听！

【2020-09-27】

【原文链接：https://xhpfmapi.zhongguowangshi.com/vh512/share/9433016?channel=weixin】

&nbsp;

声音，让世界更加生动鲜活

可是有一群人

他们的耳朵却早早“沉睡”了起来

&nbsp;

他们能看到夕阳落下

但是听不到倦鸟归巢的婉转啁啾

他们能闻到饭菜香气

但是听不到厨房里的叮当作响

他们能感到风的触碰

但是听不到风的窸窣碎语

&nbsp;

据相关资料统计

中国听障人群规模有2780万

虽有一部分人正在用人工耳蜗

但是受限于技术瓶颈

即使戴上了人工耳蜗

他们能“听到”却难“听清”

&nbsp;

现在，AI科技的进步和技术的迭代

唤醒了更多“沉睡的耳朵”

让听障者听到的声音

不再机械，更有温度

&nbsp;

晓婷

热爱舞蹈的先天性耳聋女孩

终于不用再靠着鼓点震动寻找节奏

戴上新一代人工耳蜗后

她的世界里从此有了鲜活的声音

她说：“一切比我想象中更好听”

&nbsp;

科技向善

越来越多的科技公益

正不断用技术改善这个世界

让越来越多的温暖

传递给每一个人
